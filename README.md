# CSC 680 Project

This is our project. 

## Setup 

1. Checkout this repository using git. 

    ```
    git clone git@github.com:mike-matera/csc680-project.git
    ```

1. Install NodeJS (Version 14) 

    https://nodejs.org/en/

1. Install Node dependencies. 

    ```
    cd csc680-project 
    npm install 
    ```
1. Run the Next server 

    ```
    npx next dev 
    ```

    > Browse to http://localhost:3000/
